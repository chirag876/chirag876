# 👨‍💼 About Me:
Hello! I'm Chirag Gupta, a passionate and hands-on Software Engineer with deep roots in data engineering, analytics, and full-stack application development. I specialize in transforming complex datasets into meaningful solutions that enable informed decision-making and measurable business outcomes.

With over 3 years of experience working across cloud-native and enterprise-grade environments, I’ve delivered impactful projects from building robust ETL pipelines to designing interactive dashboards and developing scalable applications using .NET and Python.

Data Engineering & ETL
I have designed and implemented end-to-end ETL pipelines using Google BigQuery, GCP, MySQL, and Python. These include schema replication, transformation logic, and dynamic resource creation. Key projects like the Pulse Billing ETL Service and BigQuery Migration Pipeline reflect my ability to handle modular architecture, batch processing, and resilient job tracking with serverless technologies like Cloud Functions.

Data Visualization & Business Intelligence
I’m skilled in translating raw datasets into compelling insights using Power BI, Tableau, and Qlik Sense. My dashboards, built for insurance and IMS domains, integrate with AWS S3, Azure, and Databricks. I’ve configured automated data refresh schedules to ensure real-time, data-driven decision-making across teams.

Python Projects & Utilities
As a Python enthusiast, I’ve developed:
• Mock Data Generator – A FastAPI-based tool that generates realistic test data with support for 12+ domains and exports in multiple formats.
• ERGenix – A Flask-based utility that creates ER diagrams and table-level insights from databases like MySQL and PostgreSQL, with export options and session security.

.NET & Full-Stack Development
I have built enterprise-grade web apps like:
• TimeTrack – A time logging system with RBAC, analytics dashboards, and project-billing workflows, developed using .NET and SQL Server.
• ShopperNet – A complete e-commerce web app with product management, role-based access, and dynamic cart features, built using ASP.NET MVC.

Tech Stack & Tools
Proficient in C#, Python, Fast API, GCP, AWS, Azure, SQL Server, MongoDB, PostgreSQL, Docker, Postman, Swagger, Entity Framework, and more. I believe in building scalable, modular, and maintainable systems with strong architectural foundations.

I'm constantly learning and evolving with emerging technologies. Let’s connect to explore opportunities to collaborate, innovate, and build solutions that create meaningful impact.
